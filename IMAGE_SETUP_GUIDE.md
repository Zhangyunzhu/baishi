## 🔧 可用的npm命令

```bash
# 启动开发服务器
npm run dev

# 下载新图片（如果需要更新）
npm run download-images

# 填补缺失图片
npm run fill-images

# 完整设置图片（下载+填补）
npm run setup-images

# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 🔄 如需更换图片

如果您有齐白石的真实作品图片，只需：

1. 将图片按照命名规范保存到对应文件夹
2. 确保图片格式为JPG或PNG
3. 推荐尺寸：
   - 藏品图片：800x600px 或更高
   - 轮播图片：1600x900px 或更高
   - 故事配图：800x600px 或更高
