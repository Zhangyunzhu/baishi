# 图片使用指南

## 📁 文件夹结构
```
public/images/
├── artworks/     # 藏品图片
├── hero/         # 轮播图片
├── stories/      # 故事配图
└── README.md     # 本文件
```

## 📝 使用方法

1. **准备图片**：将您的齐白石作品图片按照上述命名规范保存到对应文件夹
2. **推荐格式**：JPG或PNG格式
3. **推荐尺寸**：
   - 藏品图片：800x600px 或更高分辨率
   - 轮播图片：1600x900px 或更高分辨率
   - 故事配图：800x600px 或更高分辨率

## 🚀 部署说明

当网站部署时，这些图片会自动包含在构建包中，通过 `/images/` 路径访问。

例如：`/images/artworks/artwork-001.jpg`
