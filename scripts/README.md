# 🖼️ 图片爬取脚本使用指南

## 📋 脚本说明

### 1. Python爬虫 (`image-crawler.py`)
功能强大的Python爬虫，可以从必应搜索结果中自动下载齐白石画作图片。

#### 安装依赖
```bash
pip install requests beautifulsoup4 pillow
```

#### 运行爬虫
```bash
cd scripts
python image-crawler.py
```

### 2. Node.js下载器 (`download-images.js`)
简单的Node.js脚本，从预设的URL列表下载图片。

#### 运行下载器
```bash
cd scripts
node download-images.js
```

## ⚠️ 注意事项

1. **版权问题**：下载的图片仅供学习和展示使用，请注意版权问题
2. **网络限制**：某些网站可能有反爬虫机制，请适当调整请求频率
3. **图片质量**：脚本会自动过滤尺寸过小的图片
4. **备用方案**：如果爬虫失败，可以手动下载图片到对应文件夹
